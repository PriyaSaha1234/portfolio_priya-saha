## Installation

Clone the repository:

```bash
git clone https://github.com/PriyaSaha1234/Portfolio.git
```

Move into the project directory:

```bash
cd Portfolio
```

Install dependencies:

```bash
npm install
```

Run locally:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
